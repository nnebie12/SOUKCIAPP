# Play Console - Champs a remplir pour SoukCI

Ce document liste les champs Play Console avec une proposition concrete basee sur l'etat actuel du projet.

## Informations principales

- Nom de l'application: SoukCI
- Langue par defaut: francais
- Description courte suggeree: Decouvrez les boutiques et commerces de Côte d'Ivoire autour de vous.
- Description complete suggeree: SoukCI vous aide a trouver des boutiques locales, consulter leurs informations, enregistrer vos favoris, publier des avis et contacter les commercants. Selon les fonctionnalites actives, vous pouvez aussi initier des commandes et demandes de mise en avant commerciale. La geolocalisation permet d afficher les commerces proches. Les parcours de reglement affiches dans l application correspondent actuellement a des preferences de reglement et a des workflows manuels avec les commercants.
